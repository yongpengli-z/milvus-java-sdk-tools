# DescribeAliasParams

查看别名详细信息。对应组件：`custom.components.DescribeAliasComp`

## 参数

| 字段 | 类型 | 必填 | 默认值 | 说明 |
|------|------|:----:|--------|------|
| `databaseName` | String | 否 | `""` | |
| `alias` | String | 是 | | 要查看的别名 |

## JSON 示例

```json
{"DescribeAliasParams_0": {"alias": "my_alias"}}
```
