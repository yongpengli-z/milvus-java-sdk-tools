# RenameCollectionParams

重命名 Collection。对应组件：`custom.components.RenameCollectionComp`

## 参数

| 字段 | 类型 | 必填 | 默认值 | 说明 |
|------|------|:----:|--------|------|
| `collectionName` | String | 否 | `""` | |
| `newCollectionName` | String | 是 | | 新名称 |
| `databaseName` | String | 否 | `""` | |

## JSON 示例

```json
{"RenameCollectionParams_0": {"newCollectionName": "new_name"}}
```
