# GetLoadStateParams

查询 Collection 的加载状态。对应组件：`custom.components.GetLoadStateComp`

## 参数

| 字段 | 类型 | 必填 | 默认值 | 说明 |
|------|------|:----:|--------|------|
| `collectionName` | String | 否 | `""` | 为空使用最后一个 collection |
| `databaseName` | String | 否 | `""` | |

## JSON 示例

```json
{"GetLoadStateParams_0": {}}
```
