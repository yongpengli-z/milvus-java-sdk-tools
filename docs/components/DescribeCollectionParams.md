# DescribeCollectionParams

查看 Collection 详情（schema、索引信息等）。对应组件：`custom.components.DescribeCollectionComp`

## 参数

| 字段 | 类型 | 必填 | 默认值 | 说明 |
|------|------|:----:|--------|------|
| `collectionName` | String | 否 | `""` | |
| `databaseName` | String | 否 | `""` | |

## JSON 示例

```json
{"DescribeCollectionParams_0": {}}
```
