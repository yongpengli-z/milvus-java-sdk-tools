# HasCollectionParams

检查 Collection 是否存在。对应组件：`custom.components.HasCollectionComp`

## 参数

| 字段 | 类型 | 必填 | 默认值 | 说明 |
|------|------|:----:|--------|------|
| `collectionName` | String | 否 | `""` | 为空使用最后一个 collection |
| `databaseName` | String | 否 | `""` | |

## JSON 示例

```json
{"HasCollectionParams_0": {}}
```
