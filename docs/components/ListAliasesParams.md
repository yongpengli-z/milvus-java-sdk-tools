# ListAliasesParams

列出 Collection 的所有别名。对应组件：`custom.components.ListAliasesComp`

## 参数

| 字段 | 类型 | 必填 | 默认值 | 说明 |
|------|------|:----:|--------|------|
| `collectionName` | String | 否 | `""` | 为空使用最后一个 collection |
| `databaseName` | String | 否 | `""` | |

## JSON 示例

```json
{"ListAliasesParams_0": {}}
```
